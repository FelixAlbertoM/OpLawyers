OpLawyers - Sistema de Gestión Legal
¿Qué hace el proyecto?
OpLawyers es un sistema web para gestionar un bufete de abogados. Permite administrar clientes, casos legales, citas y horarios de atención.
Funcionalidades Principales
Gestión de Clientes
•	Registrar nuevos clientes con nombre, apellido, correo 
•	Ver listado de todos los clientes
•	Editar información de clientes
•	Bloquear/desbloquear clientes (impide su acceso al sistema)
Gestión de Casos Legales
•	Clientes pueden:
o	Ver todos sus casos y su estado
o	Consultar el historial de cada caso
•	Administradores pueden:
o	Crear nuevos casos describiendo su problema legal
o	Ver todos los casos del bufete
o	Agregar actualizaciones y seguimiento a los casos
o	Cambiar el estado de los casos (Abierto, En Proceso, Cerrado)
Gestión de Citas
•	Los clientes pueden agendar citas seleccionando fecha y horario disponible
•	El sistema solo muestra horarios libres para la fecha seleccionada
•	Los clientes pueden cancelar sus citas
•	Las citas canceladas liberan automáticamente el horario
•	No se permite agendar citas los domingos
Gestión de Horarios
•	Los administradores configuran los bloques de horario disponibles
•	Cada horario tiene: día de la semana, hora inicio, hora fin
•	Se pueden crear horarios predeterminados automáticamente (Lun-Vie, 8AM-5PM)
•	Los horarios se bloquean automáticamente cuando se agenda una cita
Tipos de Usuario
Administrador
•	Gestiona clientes, casos, citas y horarios
•	Ve toda la información del bufete
Cliente
•	Agenda y gestiona sus citas
•	Consulta el estado de sus casos

Cómo Funciona
1.	Registro/Login: Los usuarios se registran o inician sesión
2.	Admin: Describe su problema legal, se asigna automáticamente a su usuario
3.	Cliente agenda cita: Selecciona fecha y horario disponible
4.	Sistema bloquea horario: El horario queda ocupado para esa fecha
5.	Admin completa cita: Marca la cita como completada después de la reunión
Características Especiales
•	Bloqueo automático de clientes: Si un admin bloquea un cliente, el sistema lo desloguea inmediatamente
•	Validación de horarios: No permite duplicar horarios ni agendar en horarios ocupados
